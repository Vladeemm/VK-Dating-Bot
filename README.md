# VK Dating Bot

VK Dating Bot — модульный Python-бот для ВКонтакте, который помогает пользователям искать анкеты по критериям и сохранять понравившиеся профили в избранное.

## Структура проекта

- `main.py` — минимальная точка входа для запуска бота.
- `config.py` — загрузка конфигурации из `.env` и проверка обязательных переменных.
- `bot/core/` — основной цикл и маршрутизация событий.
- `bot/database/` — SQLAlchemy-модели, сессии и репозитории.
- `bot/vk_api/` — клиент VK, поиск анкет, фото и работа с пользователями.
- `bot/ui/` — клавиатуры, сообщения и форматирование текста.
- `bot/user/` — бизнес-логика предпочтений и избранного.

## Быстрый запуск

1. Скопируйте `.env.example` в `.env`.
2. Установите зависимости:
   ```bash
   python -m pip install -r requirements.txt
   ```
3. Заполните в `.env` значения:
   - `BOT_TOKEN`
   - `MY_VK_TOKEN`
   - `DATABASE`
4. Запустите бота:
   ```bash
   python main.py
   ```

## Примечание

Конфигурация бота проверяется при старте. Если переменные окружения не заданы, приложение завершит работу с ошибкой.
